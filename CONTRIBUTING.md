## How to Contribute
1. Fork the repository
2. Create a new branch: `feature/your-feature`
3. Make your changes
4. Submit a Pull Request

## Guidelines
- Write clean C# code
- Test before committing
- Keep commits small and meaningful

## Bug Reports
Open an issue and describe:
- What happened?
- Steps to reproduce
- Your environment (OS, .NET version)
